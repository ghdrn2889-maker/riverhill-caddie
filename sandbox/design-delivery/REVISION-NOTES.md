# 디자인 수정·결정 로그

> 사용자와 디자인 AI가 확정한 내용을 구현 AI에게 전달하기 위한 누적 문서다.

## 2026-07-18

### 다크모드 연기

- 메인 라이트 화면과 상태 설계를 먼저 완성한다.
- 다크모드는 이번 구현 범위에서 제외하고 추후 별도 작업으로 검토한다.
- 구현 AI는 이번 인계물을 이식하면서 다크모드를 임의로 추가하지 않는다.
- `index.html`은 라이트 테마로 고정했고 `states.html`은 다크모드 CSS 없이 다시 작성했다.

### 근무 확정과 티오프의 관계 수정

- **근무 확정은 티오프 배정이 이미 매칭된 상태**다.
- 근무 확정이면 티오프 시각과 코스가 반드시 함께 잡힌다.
- `근무 확정 + 티오프 아직 없음`은 실제 업무 상태가 아니므로 삭제했다.
- 상태 레퍼런스를 다음 6개로 수정했다.

  1. 근무 확정 + 티오프 있음
  2. 스페어 대기
  3. 지금 차례
  4. 휴무
  5. 정보 없음
  6. 로딩 / 오류

### 구현 AI가 갱신해야 할 원격 계약

`FRONTEND-CONTRACT.md`와 기존 인계서에서 `근무 확정 + 티오프 아직 없음`을 필수 상태로 적은 부분을 삭제하고 아래 업무 규칙을 반영한다.

> 근무 확정 상태는 티오프 시각·코스 매칭 완료를 의미한다. 근무 확정인데 티오프가 없는 별도 UI 상태는 만들지 않는다.
