# 리버힐 캐디 일정 알림

리버힐 골프장 네이버 카페(전체글)를 감시해서, **3부 / 당일추가 / 배치표** 관련 새 글이
올라오면 폰으로 즉시 알림을 보내주는 프로젝트.

- 감시: 카페 내부 JSON API를 가볍게 폴링 (헤드리스 브라우저 X)
- 알림: (예정) PWA 웹푸시 — 앱이 꺼져 있어도 알림 옴
- 구조: **운영자 계정 하나로 읽고 → 구독한 모든 캐디에게 푸시** (나중에 사업화 대비)

## 현재 단계: 스파이크(검증)

가장 위험한 부분 — "쿠키로 카페가 실제로 읽히는가" — 부터 확인합니다.

### 1) Node 확인 (18 이상)
```powershell
node -v
```
없으면 https://nodejs.org 에서 LTS 설치.

### 2) 쿠키 얻는 법 (PC 크롬 기준)
1. PC 크롬에서 **네이버 로그인** (리버힐 카페 회원 계정)
2. `F12` → 상단 **Application(애플리케이션)** 탭
3. 왼쪽 **Cookies → https://www.naver.com** 클릭
4. 목록에서 **`NID_AUT`** 와 **`NID_SES`** 의 Value 를 각각 복사
5. `.env.example` 을 복사해 `.env` 를 만들고 값 붙여넣기
   ```powershell
   Copy-Item .env.example .env
   ```
   > ⚠️ `.env` 는 로그인 세션이라 **절대 공유 금지**. (git에도 안 올라가게 설정돼 있음)

### 3) 실행
```powershell
npm run spike
```

**성공하면** 최신 글 20건 제목이 뜨고, 키워드에 걸린 글엔 🔔 표시가 붙습니다.
그 목록을 저(클로드)에게 보여주시면, 실제 제목 패턴에 맞게 키워드/필터를 정밀하게 잡고
다음 단계(24시간 크롤러 → 웹푸시 → PWA)로 진행합니다.

**실패하거나 이상한 응답이 뜨면**, 화면에 출력된 내용을 그대로 붙여주세요. 바로 잡겠습니다.
(대안: 카페 페이지에서 `F12 → Network` 탭을 열고 목록을 새로고침했을 때 뜨는
`ArticleList...` 요청의 URL을 알려주셔도 됩니다.)
